# 画面遷移図

---

## 1. 未ログイン

- `/`
  - ログイン画面へ遷移（Laravel Breeze）

---

## 2. 会員（ログイン後）

### 2.1 ダッシュボード
- `/dashboard`
  - 大会一覧 → `/tournaments`
  - 成績 → `/results`
  - 段位申請 → `/rank-requests`
  - 段位申請履歴 → `/rank-requests/history`
  - 年間登録更新 → `/membership/renew`
  - プロフィール → `/profile`

---

### 2.2 大会
- `/tournaments`
  - 大会詳細 → `/tournaments/{id}`

- `/tournaments/{id}`
  - エントリー（POST）
  - ダッシュボードへ戻る

---

### 2.3 成績
- `/results`
  - ダッシュボードへ戻る

---

### 2.4 段位申請
- `/rank-requests`
  - 段位選択・申請
  - 完了後 → `/dashboard`

---

### 2.5 段位申請履歴
- `/rank-requests/history`
  - 表示内容：
    - 申請段位
    - ステータス
    - 担当者
    - 日付（YYMMDD）
    - コメント
  - 段位申請へ戻る → `/rank-requests`
  - ダッシュボードへ戻る → `/dashboard`

---

### 2.6 プロフィール
- `/profile`
  - 日本語UI
  - 現在の段位表示
  - 更新後は同画面に戻る

---

## 3. 管理者（ログイン後 + is_admin）

### 3.1 管理者ダッシュボード
- `/admin`
  - 段位申請管理 → `/admin/rank-requests`
  - 大会管理 → `/admin/tournaments`

---

### 3.2 段位申請管理
- `/admin/rank-requests`
  - 未処理申請：
    - コメント入力
    - 承認 / 却下
  - 処理済み申請：
    - 履歴表示（担当者 / 日付 / コメント）
  - 管理者ダッシュボードへ戻る → `/admin`

---

### 3.3 大会管理
- `/admin/tournaments`
  - 大会一覧
  - 作成 / 編集 / 削除
  - 成績入力 → `/admin/tournaments/{id}/results`
