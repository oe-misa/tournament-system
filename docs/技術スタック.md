# 技術スタック・構成

---

## 1. 全体方針
- 初期は Laravel（Blade）で高速に構築
- 将来的に Next.js + React へ段階的移行可能な構成
- 認証・データ設計は API 化を前提に設計

---

## 2. 使用技術

### バックエンド
- PHP 8.x
- Laravel 12.x
- Laravel Breeze（認証）
- Laravel Sanctum（将来 API 用）

### フロントエンド
- Blade（現状）
- Tailwind CSS
- Alpine.js（Breeze標準）

### データベース
- MySQL
- Eloquent ORM

### 開発環境
- macOS
- VSCode
- Homebrew
- Composer
- Node.js / npm
- Vite

---

## 3. 認証・権限制御

### 認証
- セッション認証（Web）
- `auth` middleware

### 管理者判定
- users.is_admin = true
- admin middleware で制御
